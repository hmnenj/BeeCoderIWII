Variáveis

Vamos Aprender!

Em JavaScript, uma variável é um contêiner para armazenar dados. Ela é um nome simbólico para um valor, e esse valor pode ser alterado ao longo do tempo. Variáveis são amplamente utilizadas em programação para armazenar informações que podem ser manipuladas e referenciadas no código.
Variáveis locais e globais são conceitos fundamentais em programação que definem o escopo em que as variáveis são acessíveis dentro de um programa em JavaScript.
As variáveis locais são declaradas dentro de um bloco específico, como uma função ou um bloco condicional, e só podem ser acessadas dentro desse bloco onde foram declaradas.
Variáveis globais são aquelas declaradas fora de qualquer bloco específico, como funções, loops ou estruturas condicionais, em um programa JavaScript. Elas são acessíveis em todo o programa, o que significa que podem ser utilizadas e modificadas em qualquer parte do código, independentemente do contexto.

Tipos de Variáveis

Tipos de Variáveis

Var

Em JavaScript, var é uma palavra-chave utilizada para declarar variáveis. A variável declarada com var tem um escopo de função ou global, o que significa que ela pode ser acessada de qualquer lugar dentro da função em que foi declarada ou do escopo global, se não estiver dentro de uma função. No entanto, as variáveis declaradas com var têm algumas peculiaridades em relação ao escopo, que podem levar a comportamentos inesperados em certas situações.

Const

Em JavaScript, const é uma palavra-chave utilizada para declarar variáveis com valores constantes, ou seja, uma vez atribuído um valor a uma variável declarada com const, esse valor não pode ser alterado posteriormente durante a execução do programa. É uma forma de criar variáveis imutáveis em JavaScript. O escopo de uma variável declarada com const é semelhante ao escopo de uma variável declarada com let. Ela é limitada ao bloco em que foi declarada. Isso significa que a variável const só é acessível dentro do bloco em que foi definida.

Let

Em JavaScript, let é uma palavra-chave utilizada para declarar variáveis que são acessíveis apenas dentro do bloco em que foram definidas.Variáveis declaradas com let têm um escopo de bloco, o que significa que elas só podem ser acessadas dentro do bloco em que foram definidas, incluindo blocos de função, loops for, while, blocos if e outros blocos de código delimitados por chaves {}

Exercício 1

Qual é a principal diferença entre var, let e const em JavaScript?


a) Var é usada para declarar variáveis de escopo local, enquanto let e const são usadas para variáveis de escopo global.
b) Var tem escopo de bloco, enquanto let e const têm escopo de função.
c) Var permite a reatribuição de valores, enquanto let permite apenas a declaração de variáveis imutáveis.

a) Uma variável declarada com const não pode ser reatribuída após a sua inicialização.
b) Const é usada apenas para declarar constantes numéricas.
c) Variáveis declaradas com const têm escopo global.

Exercício 3

Qual das seguintes palavras-chave é usada para declarar variáveis com escopo de bloco em JavaScript?


a) Var
b) Let
c) Const

Exercício 4

Por que é considerado uma boa prática evitar o uso de var em favor de let ou const em JavaScript?


a) Porque var é mais eficiente em termos de desempenho.
b) Porque var é uma palavra-chave obsoleta que não é mais suportada em navegadores modernos.
c) Porque var pode levar a problemas de escopo e dificultar a manutenção do código.
