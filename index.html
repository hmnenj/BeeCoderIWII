Sobre o JavaScript

Java Script

JavaScript é uma linguagem de programação de alto nível, interpretada e dinâmica, usada para tornar as páginas web interativas e dinâmicas. É executada no navegador do usuário e permite adicionar comportamento dinâmico aos elementos HTML, interação com APIs e serviços web, e construção de aplicativos web completos tanto no lado do cliente quanto no lado do servidor com tecnologias como Node.js.

HTML 5

HTML5 é a versão mais recente da linguagem de marcação utilizada para estruturar e apresentar conteúdo na web. Introduziu novos elementos semânticos, suporte embutido para áudio e vídeo, formulários mais poderosos e APIs para armazenamento local e offline.

CSS 3

CSS3 é a última versão da linguagem de estilo usada para controlar a apresentação e o layout dos elementos HTML em uma página web. Introduziu diversas novas propriedades e funcionalidades, como gradientes, sombras, transições, animações e flexbox, permitindo designs mais criativos e responsivos
