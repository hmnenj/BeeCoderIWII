Originada como uma linguagem de script simples para interatividade em páginas web, JavaScript evoluiu para uma linguagem poderosa e versátil, capaz de criar desde simples interações de usuário até complexas aplicações web e até mesmo aplicativos de servidor.

Como Imprimir JS no HTML

Alert

Em JavaScript, o alert() é uma função embutida que exibe uma caixa de diálogo modal com uma mensagem para o usuário. É comumente usado para fornecer informações simples ou alertas ao usuário em páginas da web. A caixa de diálogo exibe o texto fornecido como argumento para a função alert().

Prompt

O prompt() é uma função embutida em JavaScript que exibe uma caixa de diálogo modal com uma mensagem para o usuário e permite que o usuário insira um valor. É útil quando você precisa coletar dados ou entrada do usuário em uma página da web.

Confirm

Em JavaScript, o confirm() é uma função embutida que exibe uma caixa de diálogo modal com uma mensagem e botões "OK" e "Cancelar". Ele é amplamente utilizado para solicitar uma confirmação do usuário em páginas da web.

Documente.write

Em JavaScript, o document.write() é um método embutido que permite escrever conteúdo diretamente no documento HTML a partir do código JavaScript. Ele é usado para adicionar conteúdo dinamicamente à página enquanto ela está sendo carregada ou durante a execução do script.

Tipos de Dados

String

Uma sequência de caracteres, normalmente utilizada para representar texto. As strings são delimitadas por aspas simples ('') ou duplas ("").
Exemplo: "Olá, mundo!"

Number

Representa números, incluindo números inteiros e de ponto flutuante (decimais). Não há distinção entre números inteiros e de ponto flutuante em JavaScript; ambos são representados pelo tipo Number.
Exemplo: 10, 3.14

Boolean

Representa um valor lógico, que pode ser true (verdadeiro) ou false (falso). É comumente usado em operações de lógica e controle de fluxo.
Exemplo: true, false

Array

Array: Um objeto especial que contém uma coleção ordenada de elementos. Os elementos podem ser de qualquer tipo de dado e são acessados por um índice numérico.
Exemplo: [1, 2, 3, 4, 5]

Function

Uma função em JavaScript é um objeto que executa um bloco de código quando chamado.
Exemplo: function soma(a, b) {
return a + b;
}
